# Environment Stock Images

This folder contains stock images for each game environment. These images are displayed to players to help them visualize their survival scenario.

## Required Images

Place the following stock images in this folder:

- **coast.jpg** - Rocky coastline with waves (for coast environment)
- **mountains.jpg** - Mountain landscape (for mountains environment)
- **desert.jpg** - Desert landscape (for desert environment)
- **forest.jpg** - Dense forest (for forest environment)
- **tundra.jpg** - Arctic/tundra landscape (for tundra environment)
- **urban-edge.jpg** - Urban/wilderness boundary (for urban-edge environment)

## Image Specifications

- **Format**: JPG or PNG
- **Dimensions**: 1200x600px or similar landscape aspect ratio (2:1)
- **Size**: Keep under 500KB for fast loading
- **Style**: Realistic, atmospheric, suitable for survival scenarios
